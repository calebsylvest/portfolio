Pattern Tap

I love Pattern Tap because they offer great resources to Web Designers. Examples of good web design are offered in convenient catagories including Navigation, Headers, Footers, Search Bars and much much more.

 

Flash Game Tuts

If you are interested in getting into Flash game development and only know a little about Action Script, I would suggest Flash Game Tuts, some of the most well written tutorials. And in good English too!

I Heart Movies allows you to keep track of and rate movies that you have seen, and also see top rated movies of others. And yes, I know Netflix has this also. I don't subscribe to Netflix.

 

Pixel Purge

If I ever get annoyed while pushing pixels I take a little trip over to Pixel Purge land and blast away. Feels good everytime.

This is the game for Web Designers. Not because of the gameplay but because the game is built entirely using Javascript and HTML5. But it is still cool, I wish this game existed when I was younger, I would be a better typist.
